Beatmap API

Usage:
                          /   Returns this page.
                              
              /[0-9a-f]{32}   Returns the beatmap or song folder json data with the given hash.
                              Beatmaps always have the same hash. Song folders may differ.
                              Song folder hashes can be retrieved through the given beatmap.
                              
      /[a-f0-9]{32}(/args)?   Returns the beatmap or song folder data with the given arguments.
                              'args' can be any of the following
                                     R - Raw data
                              R/ - Raw data of a given file
                                     A - Audio file
                                     S - Storyboard file
